# MindConnect+ Mental Health Support Platform

## Project Overview
A comprehensive mental health support application with mood tracking, 
peer matching, support groups, and counseling sessions.

## Team Members
- Sravya
- Ehsan
- Rajeshwari 

## Technology Stack
- **Database:** Oracle Database 19c
- **Frontend:** Streamlit (Python)
- **Backend:** Python 3.8 with oracledb driver
- **Visualization:** Pandas, Matplotlib

## Project Phases

### Phase I: Database Design
- ER Diagram
- Relational Schema

### Phase II: Implementation
- DDL statements (table creation)
- Sample data insertion
- Integrity constraints

### Phase III: Operations
- DML operations
- Complex queries with JOINs, aggregates, subqueries

### Extended Phase III: Application
- Full-stack web application
- User-friendly interface
- Real-time database operations

## Quick Start

### Prerequisites
- Python 3.8+
- Oracle Database access
- Required Python packages

### Installation
```bash
# Install dependencies
pip install -r requirements.txt

# Configure database connection
# Edit app.py with your credentials

# Run application
streamlit run Extended_Phase3_Application/app.py
```

## Database Setup
Connect to the oracle db and execute python test-connection.py from extended phase3 app folder
